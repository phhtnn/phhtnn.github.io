Validating Forms (with React)

Tags: Form validation React

I think that trying to build form validations from scratch helped me learn a lot about web development. I’d do many things differently now, but here’s a couple things I learned during the process. First, I think that I understand the different roles of frontend, backend and database dependant validation much better than a couple of months ago. Second, at this point I’m pretty much agreeing with the idea that in most cases, the form state in React is best managed locally rather than globally with e.g. Redux. Third, one could code the frontend validation in just React (e.g. like this), but using a library like Formik probably helps to keep all the things organized.
